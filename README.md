# Health Dashboard API

A Spring Boot 3 backend for personal health metric tracking — a portfolio project demonstrating REST API design, JPA persistence with PostgreSQL, JWT authentication, and modern Java practices. Currently in early development; the eventual goal is a deployed API supporting CRUD operations, bulk data import, and aggregation endpoints over personal Fitbit and supplement data.

## Tech Stack
- Java 21
- Spring Boot 3.5
- Spring Data JPA
- PostgreSQL
- Maven
